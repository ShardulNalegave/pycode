
# Pycode
Code Editor


# What Is PyCode?
**PyCode** is a code editor made using ***WxPython** And **Python 3***.


# Usage
Clone this repo and run the `main.py` file.

````bash

    git clone https://github.com/ShardulNalegave/pycode.git PyCode
    cd PyCode
    python[x[.y]] main.py

````
